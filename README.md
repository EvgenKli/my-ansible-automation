# Configuration Management & Kubernetes Bare-Metal Cluster Deploy

Репозиторий содержит автоматизированные сценарии управления конфигурацией серверов с помощью Ansible.

## Развернутые сценарии (Playbooks)
1. **`deploy_postgres.yml`** — Идемпотентный сценарий установки и конфигурации СУБД PostgreSQL 16. Настраивает сетевой доступ (`pg_hba.conf`), создает защищенных пользователей и базы данных.
2. **`install_k8s_tools.yml`** — Сценарий подготовки нод и развертывания ядра Kubernetes. Автоматизирует отключение Swap, настройку сетевых фильтров ядра Linux, установку контейнерного движка `Containerd` (с поддержкой Systemd Cgroup) и утилит `kubeadm/kubelet/kubectl`.

## Сетевые особенности
Реализована схема подключения к серверам в закрытом контуре через Бастионный хост (Jump Host) с использованием SSH ProxyCommand. Настроена беспарольная автоматическая связка ключей между нодами кластера.
